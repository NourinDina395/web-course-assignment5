1️⃣ What is the difference between var, let, and const?
->
In JavaScript, var, let, and const are used to declare variables. var has function scope and can be redeclared and reassigned, which may cause unexpected behavior. let has block scope and can be reassigned but cannot be redeclared in the same scope. const also has block scope but cannot be reassigned after its initial declaration, making it useful for values that should remain constant.

2️⃣ What is the spread operator (...)?
->The spread operator (...) is used to expand elements of an array or properties of an object into another array or object. It is commonly used to copy arrays or objects, merge them, or pass elements as individual arguments to a function.


3️⃣ What is the difference between map(), filter(), and forEach()?
->map(), filter(), and forEach() are array methods used to iterate over elements. map() creates a new array by transforming each element, filter() creates a new array containing only elements that meet a specific condition, and forEach() simply executes a function for each element without returning a new array.

4️⃣ What is an arrow function?
->An arrow function is a shorter syntax for writing functions in modern JavaScript. It uses the => symbol and allows developers to write more concise code, especially for small functions and callbacks.

5️⃣ What are template literals?
->Template literals are a way to create strings in JavaScript using backticks (`). They allow embedding variables or expressions directly inside the string using ${}, making string formatting and concatenation easier and more readable.
